# 数学分析

复数由实数定义。实数集是复数集的子集。复分析以自然数为先验概念，构建出体系。

复分析的一些定理与实分析形式相似，一些证明又需要对实虚部二元实函应用实分析定理。

复分析是信号和系统分析的有效工具。
